What it is like as a student of technology

Learning to code was to me just like learning a whole new language. I have not really tried coding before starting with this bachelor program at OsloMet. During my high school academic years, I never really had any classes about coding or information technology either. I always thought that coding need time to be familiar with and practice smoothly and without any difficulty. I think that experimenting with examples and functionality would improve our skills and competence.

What encouraged me to take the course was that I read very positive experiences and reviews regarding this career and working as a coder and similar jobs. Information technology was never a field that I have ever thought I would pursue for my career life but that is how my life suddenly turned out to revolve around. Coding encompasses many dissimilar languages that would demand time and patience in order to master. I think that this bachelor program presents many of these different coding languages, but it could be so that not all are within our learning syllabus.

During this first semester we learn about programming through using Java which I think is way difficult than html and CSS. I do not think though that these languages are the ones I will work mostly with later on in life, but it is still an option. What could be very challenging with coding in general is that the techniques that could innumerous. I think it could be true that every single minute there would be a new functionality for coding that is been created. It is still exciting to imagine the number of opportunities that such knowledge can open up. I still do not know exactly what I will be working with within this field taking into consideration how versatile this industry is.

challenges of studying computer technologies

Mentioning programming and Java language specifically as an instance, it is absurd how these techniques could be handled. Another serious challenge while coding is being capable to fix issues within the codes when having those. This part can be really time challenging. Firstly, it can be challenging to be able to transform ideas into a real working code. I remember spending many hours and days working on errors or wrongly formulated codes that needed to be restructured.

Speaking about coding and to be more specific about web development and design, I really had spent time to learn several features including, layout, positioning, grid, responsive design etc. In order to build all different websites that I have already built myself I have gone through all the learning that were available on the university platform and other relevant coding sites. The resources included several links into tutorial websites that would be beneficial for designing. In addition, I have spent several days observing educational tutorials on YouTube.com that explain the different functionality and designing choices that would be taken into account. The best part of this all is when everything finally works out & you are able to implement your ideas. I have always thought that designing is very creative and fun but time challenging as well. I look forward to work with such website designing projects in the future.
