Accessibility

Accessibility of a web page means how easy it is to use and how well supported it is for different sized devices. It also includes making design desissions to ensure readability, easy navigation and usability. This can be ensuring the webpage can be used by a blind person, or someone who is color blind.

We have improved readability by using good spacing and text size which scales dynamically. We have used a simple color pallet to make navigation very clear. We have also used a color contrast checkers to check the contrast on the text to background. We have built this page with responsiveness in mind. The page can be scaled, and the elements will flow nicely instead of breaking. On the mobile version we also have a different menu. A hamburger menu with a animated menu.
